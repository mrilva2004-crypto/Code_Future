# Book Tracker - Трекер прочитанных книг

**Автор:** Иван Петров

## Описание программы

Book Tracker - это графическое приложение на Python для ведения учета прочитанных книг. Программа позволяет добавлять книги с указанием названия, автора, жанра и количества страниц, фильтровать их по жанру и количеству страниц, а также сохранять данные в JSON файл.

### Возможности:

- Добавление книг с проверкой корректности ввода
- Просмотр всех книг в табличном виде
- Фильтрация книг по жанру
- Фильтрация книг по количеству страниц (> N)
- Удаление книг из списка
- Сохранение данных в JSON файл
- Автоматическая загрузка данных при запуске

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, os)
